# Customizações de Agente - ContratoBus

## Comandos de Design

### `/interface-design:init`

Inicializa ou reinicializa o sistema de design.

**Lógica:**

- Se `.interface-design/system.md` NÃO existe: Explore domínio → Proponha direção → Confirme com usuário → Construa sistema
- Se `.interface-design/system.md` JÁ existe: Leia sistema → Ofereça próximos passos ou alterações

**Saída esperada:**

- Confirmação que o sistema está pronto
- Próximas ações: `/interface-design:audit`, `/interface-design:apply`, `/interface-design:extract`

### `/interface-design:status`

Mostra o estado atual do sistema de design (lê `.interface-design/system.md`)

### `/interface-design:audit`

Audita componentes existentes contra o sistema definido

### `/interface-design:apply`

Aplica o sistema de design ao código existente (atualiza estilos, tokens, componentes)

### `/interface-design:critique`

Critica o trabalho feito contra princípios de craft e propõe melhorias
